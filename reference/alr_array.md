# Compute the ALR of an array

Compute the ALR of an array

## Usage

``` r
alr_array(x, d = dim(x)[parts], parts)
```

## Arguments

- x:

  multidimensional array in simplex

- d:

  Index of column used as a reference. Defaults to last column

- parts:

  index of dimension of \`x\` that represents parts

## Value

array
