# Generic Method to Plot Posterior Predictive Summaries

Generic Method to Plot Posterior Predictive Summaries

## Usage

``` r
# S3 method for class 'pibblefit'
ppc_summary(m, from_scratch = FALSE, ...)

ppc_summary(m, ...)
```

## Arguments

- m:

  model object

- from_scratch:

  should predictions of Y come from fitted Eta or from predictions of
  Eta from posterior of Lambda? (default: false)

- ...:

  other arguments to pass

## Value

vector
