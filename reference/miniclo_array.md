# Closure Operation applied to array on margin

Array version of
[`miniclo`](https://jsilve24.github.io/fido/reference/miniclo.md).

## Usage

``` r
miniclo_array(x, parts)
```

## Arguments

- x:

  multidimensional array

- parts:

  index of dimension of `x` that represents parts (e.g., compositional
  variables)

## Value

array

## Examples

``` r
x <- array(1:100, dim=c(10, 5, 2))
miniclo_array(x, parts=2)
```
