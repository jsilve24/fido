# Data from Gevers et al. (2014)

OTU data and metadata for 1,359 samples in a Crohn's disease study

## Usage

``` r
data(RISK_CCFA)
```

## Format

A taxonomy table.

## Details

Study is described here: <https://pubmed.ncbi.nlm.nih.gov/24629344/>.
Data was obtained from <https://github.com/twbattaglia/MicrobeDS>.

## References

Gevers D, et al. The treatment-naive microbiome in new-onset Crohn's
disease. Cell Host Microbe. 2014 Mar 12;15(3):382-392. doi:
10.1016/j.chom.2014.02.005. PMID: 24629344; PMCID: PMC4059512.
